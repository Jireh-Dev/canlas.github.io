## 🖼️ How to Add Images
1. Create a folder called `imgs` in the same folder as all .html files
2. Rename your photos to match the filenames above
3. Drop them into the `imgs/` folder
4. Open any HTML file in your browser — images will appear automatically!

> **TIP:** If an image is missing, the website still works — it just shows a placeholder.
> You can use any image filename as long as you update the `src="imgs/yourfilename.jpg"` in the HTML.
